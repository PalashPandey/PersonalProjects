Abstract Algebra


Group Theory

A group is a set of objects combined with an operation, usually denoted as (*) or (+).

Additionally, every group must satisfy four conditions:

1. Identity: There is an element '1' such that 1*a = a*1 = a for any element a in the group.
2. Closure: For any elements a and b in the group, a*b must be in the group.
3. Inverses: For any element a, there is a-1 such that a*(a-1) = (a-1)*a = 1.
4. Associativity: For any a, b, and c in the group, a*(b*c) = (a*b)*c.


Many of the mathematical objects we encounter every day are groups. Real numbers and integers form groups under addition. Permutations of a set of objects form a group under composition. Do the rational numbers form a group under multiplication?


Generating Elements

Generating elements form the basis of the set of group elements.
Every element in a group can be written as a combination multiplication and inversion of the generating elements.

For finite groups, the generating elements have finite order.
I.E. for every generating element a, there is some positive integer n such that an = 1.

If a set of generating elements is used as the edge set for a Cayley graph, every node can be reached from any other node by following the graph's edges.


Defining Equations

Defining equations describe how different generating elements interact under the group operation.
Defining equations are of the form ap * bq = bn * an.
In our traditional number systems, a*b = b*a for all numbers a and b. This property is called commutativity.


Cayley Tables and Graphs

Cayley Tables describe how the elements of a group interact, similar to a multiplication table.
If the set and operation satisfy all the requirements for a group, each row and each column will contain exactly one of each element.
Cayley graphs are another way to visualize a group.
Each node represents an element of the group, and each edge represents multiplication by a specific element.
